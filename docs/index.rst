Sphinx-sitemap
==============

|PyPI version| |Conda Forge| |Downloads| |Parallel Safe| |GitHub Stars|

A `Sphinx`_ extension to generate multi-version and multi-language
`sitemaps.org`_ compliant sitemaps for the HTML version of your Sphinx
documentation.
